Filtrado de Agua

Acerca del proyecto "filtradora"

El proyecto seleccionado : corresponde a una Filtradora de Agua. Este tiene como propósito aplicar principios básicos de electrónica y programación para crear un sistema capaz de mejorar la calidad del agua mediante procesos de filtración supervisados con sensores.

1. Diseño y conceptualización del sistema

El sistema consiste en un Arduino UNO conectado a un módulo Bluetooth HC-05 que permite la comunicación inalámbrica con una aplicación móvil desarrollada en MIT App Inventor. El microcontrolador procesa las señales recibidas desde la aplicación y controla los actuadores (en este caso, un LED). Sensores/Entradas: . En este prototipo no se utilizan sensores, la entrada proviene de la aplicación móvil (botones ON/OFF).

Problemas que soluciona

  • Optimización de condiciones ambientales: • La práctica permitió afianzar conocimientos sobre la configuración de pines analógicos, la visualización de datos en el monitor serial y la importancia de registrar las pruebas en tablas para validar los resultados.
  • Gestión eficiente de recursos: • Aunque se probó únicamente el modulo bluetooth, este avance representa un paso importante dentro del proyecto, ya que asegura que el sistema puede empezar a integrar mediciones reales que en fases posteriores se complementarán con otros sensores y con la comunicación Bluetooth.
  • Montage del proyecto • Finalmente, el montaje y las pruebas iniciales también ayudaron a identificar la necesidad de documentar todo el proceso con evidencias (fotos, capturas y repositorio en GitHub), lo cual fortalece la presentación técnica del proyecto y la claridad del trabajo colaborativo del grupo.

Énfasis en Mecatrónica

Este proyecto integra los principios de la Mecatrónica, combinando electrónica, programación y diseño mecánico para dar una solución práctica y educativa a un problema de relevancia mundial.
